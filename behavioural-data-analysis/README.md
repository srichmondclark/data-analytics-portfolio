# Behavioural Data Analysis

## Overview
This project focuses on analyzing behavioural datasets to uncover patterns and insights in human behaviour and decision-making processes.

## Project Goals
- Analyze behavioural patterns in datasets
- Identify key trends and correlations
- Generate actionable insights from behavioural data
- Apply statistical methods to understand human behaviour

## Tools & Technologies
- R for statistical analysis
- Data visualization techniques
- Exploratory data analysis (EDA)
